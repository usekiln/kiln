# Kiln
   
   SOC2 compliance scanner for Terraform
   
   ## Quick Start
```bash
   go install github.com/usekiln/kiln/cmd/kiln@latest
   kiln scan main.tf
```
   
   ## Status
   
   Early development. Not ready for production use.
